# Appetite

Fighting poverty, one donation at a time.

## First start the backend
1. `cd backend`
2. `yarn`
3. `yarn dev`

## Next start the app
1. `cd app`
2. `yarn`
3. `yarn dev`
4. Go to `http://localhost:8080`
5. Click inspect element and click the mobile icon in the top left.
6. The app should now be running.
